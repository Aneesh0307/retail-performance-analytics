# End-to-End Retail Performance & Behavioral Analytics

## Overview
This project presents a complete retail analytics solution that transforms raw transactional data into actionable business insights using Python, SQL, and Power BI.

The analysis focuses on sales performance, customer behavior, product profitability, store efficiency, and return patterns.

## Problem Statement
Retail businesses generate large volumes of data but lack unified analytics to:
- Track performance across regions and channels
- Understand customer behavior
- Identify high-performing products and stores
- Analyze return patterns

## Tech Stack
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (SQL Server)
- Power BI
- Excel

## Data Sources
- Customers
- Products
- Sales
- Stores
- Returns

## Project Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Database Design (Fact-Dimension Model)
4. SQL-Based Analysis
5. Dashboard Development (Power BI)

## Key Insights
- In-store sales generate higher revenue than online channels
- A small number of stores contribute majority of total profit
- Certain customer age groups drive higher revenue
- Product return rates vary significantly by category
- Regional differences impact sales and return behavior
